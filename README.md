# GTNX
A pack containing essential homebrew software for the Nintendo Switch.
![GTNX](https://github.com/gametekrepo/GTNX/assets/150208884/26d04ca8-13ca-4d2f-af92-a473c87131d4)

## Included software
* [Atmosphere](https://github.com/Atmosphere-NX/Atmosphere)
* [hekate](https://github.com/CTCaer/hekate)
* [DBI](https://github.com/rashevskyv/dbi)
* [AIO-Switch-Updater](https://github.com/HamletDuFromage/aio-switch-updater)
* [Homebrew App Store](https://github.com/fortheusers/hb-appstore)
* [JK's Save Manager](https://github.com/J-D-K/JKSV)
* [Linkalho](https://github.com/rdmrocha/linkalho)
* [nxdumptool](https://github.com/DarkMatterCore/nxdumptool)
* [NX-Shell](https://github.com/joel16/NX-Shell)
* [Mission Control](https://github.com/ndeadly/MissionControl)
* [Lockpick_RCM](https://github.com/s1204IT/Lockpick_RCM)
